Markdown快速入门

一、认识Markdown

Markdown是一种轻量级标记语言，创始人为约翰·格鲁伯（John Gruber）。它允许人们“使用易读易写的纯文本格式编写文档，然后转换成有效的XHTML（或者HTML）文档”。这种语言吸收了很多在电子邮件中已有的纯文本标记的特性。
Markdown的语法全由一些符号所组成，这些符号经过精挑细选，其作用一目了然。
